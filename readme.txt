# Dieta Profesional España - Streamlit App

Esta app permite generar un menú semanal editable con sustituciones automáticas,
lista de compra y descarga de Excel con toda la información nutricional.

## Cómo usarla

1. Crea una cuenta gratuita en https://share.streamlit.io
2. Sube este repositorio como un nuevo app:
   - New app → conecta tu repo → selecciona `dieta_profesional_sustituciones.py` → Deploy
3. ¡Listo! La app estará online y podrás abrirla desde cualquier navegador.
4. Ajusta tus parámetros (peso, altura, objetivo, días, etc.).
5. Cambia alimentos si quieres sustituciones automáticas.
6. Descarga tu Excel profesional con un clic.
